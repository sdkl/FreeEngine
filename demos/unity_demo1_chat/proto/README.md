目前只支持在windows下生成proto代码。
用法：双击Gen.bat即可生成相关文件并自动复制到工程目录下，点击前最后先用文本编辑核对一下目录相关信息。

c++用的是工程里的protobuf工程生成的protoc.exe。

c#用的是http://www.ceeger.com/forum/read.php?tid=13479这个链接里介绍的方法，因为如果要发布到IOS，不能动态编译代码，需要预编译。
